Tekniska Kompetenser

Java är ett populärt programmeringsspråk som används för att bygga allt från webbaserade applikationer till mobilappar. Java är ett av mina starkaste programmeringsspråk, där jag har mest erfarenhet.

SQL används för att hantera och manipulera data i databaser. Jag har kunskaper i att designa och optimera databaser med SQL.

HTML är grunden för alla webbplatser och används för att strukturera innehåll. Jag behärskar HTML för att skapa responsiva, semantiska och tillgängliga webbsidor.

CSS

JavaScript

JavaScript används för att skapa interaktiva funktioner på webbplatser. Jag har lite mindre erfarenhet i JavaScript men finner det intressant och ser fram emot att fördjupa mig i det!
